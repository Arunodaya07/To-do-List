To-Do List Web Application

A simple and responsive To-Do List web application built using **HTML, CSS, and JavaScript**.
This project helps users manage their daily tasks efficiently by allowing them to add, complete, and delete tasks through an interactive user interface. 
Tasks are stored using **Local Storage**, ensuring that they remain available even after refreshing or reopening the browser.

Features

* Add new tasks dynamically
* Mark tasks as completed
* Delete tasks when no longer needed
* Responsive and user-friendly design
* Persistent task storage using Local Storage
* Clean and intuitive interface

Technologies Used

* HTML5
* CSS3
* JavaScript

Key Concepts Applied

* DOM Manipulation
* Dynamic Element Creation (`createElement`, `appendChild`)
* Event Handling (`addEventListener`)
* Event Delegation using `e.target`
* CSS Class Manipulation (`classList.toggle`)
* Local Storage for data persistence
* Parent-Child DOM Relationships (`parentElement`)
* Responsive Web Design
